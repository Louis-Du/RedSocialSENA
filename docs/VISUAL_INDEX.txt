┌────────────── FLUJO DE DATOS EN TIEMPO REAL ────────────────────────────┐
│                                                                          │
│                                                                          │
│  Usuario A en Browser A         Firebase Firestore       Usuario B en B │
│  ═════════════════════════       ══════════════════       ═════════════ │
│                                                                          │
│      1. Empieza escribir                                                 │
│      setTyping(chatId, true) ──→ typing collection                     │
│                                  ↓                                       │
│                                  ↓ listener B escucha                   │
│                              "está escribiendo..."                      │
│                                  ↑                                       │
│      2. Envía mensaje                                                    │
│      sendMessage(...) ──→ Transacción:                                  │
│                           1. crear mensaje                               │
│                           2. actualizar lastMessage       │            │
│                           ↓                               ↓            │
│                          messages/ updates              Usuario B ve   │
│                          readBy: [A]  RT                mensajes       │
│                                  ↓                        ↓             │
│                          Listener B recibe               markRead()     │
│                          onSnapshot ────────→ Renderizar   │         │
│                                               readBy: [A,B] →          │
│                                                                          │
│      3. Deja de escribir                                                 │
│      setTyping(chatId, false) ──→ typing deleted                       │
│                                   ↓                                      │
│                                   ↓ Listener B actualiza                │
│                              <sin indicador>                             │
│                                                                          │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘


════════════════════════════════════════════════════════════════════════════════

┌──────────── COMPARATIVA: ANTES vs DESPUÉS ─────────────────────────────┐
│                                                                        │
│  ASPECTO              ANTES              │       DESPUÉS              │
│  ──────────────────────────────────────┼─────────────────────────── │
│  Fuente de datos     AppState (RAM)      │ Firestore (Cloud)         │
│  Persistencia        ❌ Solo local       │ ✅ Cloud + Offline cache   │
│  Usuarios            1 per app           │ ✅ Múltiples users        │
│  Realtime            ❌ No               │ ✅ Sí (onSnapshot)        │
│  Chat privado        Manual ID           │ ✅ ID determinístico      │
│  Chat grupal         ❌ No               │ ✅ Sí con admin tools     │
│  Imágenes            Base64 limitado     │ ✅ Storage URLs           │
│  Indicador escr.     ❌ No               │ ✅ Typing indicator       │
│  Autenticación       ❌ No validada      │ ✅ Firebase Auth          │
│  Seguridad           ❌ Baja             │ ✅ Firestore Rules        │
│  Escalabilidad       ❌ Baja             │ ✅ Miles de usuarios      │
│  Timestamps          Locales             │ ✅ serverTimestamp()      │
│  Transacciones       ❌ No               │ ✅ runTransaction()       │
│  Listeners           Ninguno             │ ✅ 3 activos coordinados   │
│  Limpieza recursos   ❌ No               │ ✅ cleanup() + destroy()  │
│                                                                        │
└────────────────────────────────────────────────────────────────────────┘

│                                                                          │
│  ChatService (Lógica Firebase)                                          │
│  ═════════════════════════════════════                                  │

│                                                                          │
│  listenToMessages(chatId, callback)                                     │
│  └─ Callback: (error, messages) ⟹ Actualiza RT                          │
│  └─ Retorna: unsubscribe function                                       │
│                                                                          │
│  listenToUserChats(callback)                                            │
│  └─ Callback: (error, chats) ⟹ Actualiza lista RT                       │
│  └─ Retorna: unsubscribe function                                       │
│                                                                          │
│  async setTyping(chatId, isTyping)                                      │
│  └─ Escribe/borra documento en typing collection                        │
│                                                                          │
│  listenToTyping(chatId, callback)                                       │
│  └─ Callback: (error, typingUserIds) ⟹ Array de UIDs escribiendo       │
│  └─ Retorna: unsubscribe function                                       │
│                                                                          │
│  async markMessageAsRead(chatId, messageId)                             │
│  └─ Agrega usuario actual a array readBy                                │
│                                                                          │
│  async addGroupParticipants(chatId, userIds)                            │
│  └─ Solo creador del grupo                                              │
│                                                                          │
│  async removeGroupParticipant(chatId, userId)                           │
│  └─ Creador o el usuario mismo                                          │
│                                                                          │
│  async getChat(chatId)                                                  │
│  └─ Obtiene chat con validación de permisos                             │
│                                                                          │
