# Documentación SQA (Software Quality Assurance)

## Propósito
Esta carpeta contiene toda la documentación relacionada con el aseguramiento de calidad del software para el proyecto Red Social del SENA. El objetivo es garantizar que el desarrollo cumpla con los estándares de calidad establecidos, identificar problemas tempranamente y asegurar que el producto final satisfaga los requisitos funcionales y no funcionales.

## Contenido actual

### 📄 Plan de SQA.pdf
Documento maestro que define:
- Objetivos y alcance del aseguramiento de calidad
- Metodologías y estándares a seguir
- Roles y responsabilidades del equipo
- Cronograma de actividades de calidad
- Métricas y criterios de aceptación

### ✅ Evaluación del comportamiento... - Listas de chequeo correccion.pdf
Listas de verificación utilizadas para:
- Evaluar el comportamiento del sistema
- Verificar requisitos funcionales
- Validar criterios de aceptación
- Documentar correcciones necesarias

### 📊 Informe sobre la lista de chequeo y evaluación.pdf
Informe consolidado que presenta:
- Resultados de las evaluaciones realizadas
- Análisis de cumplimiento de requisitos
- Identificación de no conformidades
- Recomendaciones de mejora
- Estado general de calidad del proyecto

## Documentación futura
Esta carpeta eventualmente contendrá:
- Matrices de trazabilidad (requisitos vs. pruebas)
- Reportes de auditorías de calidad
- Métricas de calidad del código
- Documentación de procesos de revisión
- Planes de mejora continua

---

**Nota:** Todos los documentos en esta carpeta son parte integral del proceso de certificación y validación del proyecto académico.
